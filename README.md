# Assistente Saúde

Personal health assistant website developed with HTML, CSS and JavaScript.

🔗 Live project: https://assistente-saude.vercel.app/

## Technologies

- HTML
- CSS
- JavaScript
- Vercel

## Purpose

This project was created to organize health-related information in a simple and accessible interface, including routines, reminders, contacts and useful personal notes.

## Features

- Calendar section
- Health and exercise section
- Medication section
- Emergency contacts section
- Notes section
- Important announcements section
- Responsive page structure
